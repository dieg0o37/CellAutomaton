# Rules Formatting
- State = "Dead" or "Alive"
- Operator = "=", ">=", "<=", ">", "<", "!="
- Int = Number of alive neighbors

```
    # Conditions for a cell to live based on their current state and the number of alive neighbors
 if <State> <Operator> <Int>
 if <State> <Operator> <Int>
    ....
    # as many as you want
```
